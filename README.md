# Fire Extinguisher - Stop the Fire

A simple fire extinguisher to stop the fire or to freeze mobs.

![][logo]

## Features

- Easy to craft
- Could be placed as block
- Removes fire from a specific radius and not only a single block
- Could freeze Mobs
